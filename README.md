# 📝 Notes App

## 📸 Screenshot

![Notes App Screenshot](https://github.com/harshkumar225/Notes-App/blob/main/Screenshot%202026-06-19%20100031.png)


A simple and efficient Notes application built with **ReactJS**.  
This app helps users create, edit, and delete notes in a clean and responsive interface.

## 📖 About the Project
The Notes App is designed to organize tasks, ideas, and reminders.  
It demonstrates modern React development practices like component-based architecture, state management, and responsive design.

## 🚀 Features
- Add new notes with title and description
- Edit existing notes
- Delete notes easily
- Responsive design for mobile and desktop
- Clean UI with React components

## 🛠️ Tech Stack
- ReactJS
- CSS
- JavaScript
- Vite (for fast development and build)
